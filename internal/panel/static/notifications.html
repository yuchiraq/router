FAQ: как запускать с Telegram на localhost
Почему на localhost бот не отвечает через webhook?

Telegram принимает webhook только на публичный https:// URL. Адреса вида http://localhost:8162 Telegram не вызывает.

Что нужно сделать, чтобы работало локально?

Поднимите публичный HTTPS endpoint (например, через reverse proxy с доменом или tunnel), затем укажите его в поле Webhook URL в формате https://ваш-домен/telegram/webhook и нажмите Save Notification Settings.

Минимальные шаги настройки
  1. Заполните Token и Chat IDs.
  2. Оставьте или задайте Webhook Secret.
  3. Укажите публичный Webhook URL с https://.
  4. Сохраните настройки и отправьте тестовое сообщение кнопкой Send Test Message.
